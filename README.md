## Usage

-   Run [ganache-cli](https://github.com/trufflesuite/ganache-cli)
-   Change into `projek-truffle-bahagian-2`
-   To migrate smart contract:<br />
    \$ truffle migrate
-   To test:<br />
    \$ truffle test
-   To interact via console:<br />
    \$ truffle console
